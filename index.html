THE BEAT MACHINE

A=[..,.]B=[?.?.]2[A]B

This is a little beat machine! type a beat code in the text box! each punctuation mark is a different beat. There's also loops and variable assignments

  • . strong beat
  • , light beat
  • : double beat
  • ? rest 1 beat
  • N[...] repeats the bracketede section N times
  • var_name=[...] assign a sequence to a variable. semicolon marks the end of assignment
  • var_name de-reference a variable
